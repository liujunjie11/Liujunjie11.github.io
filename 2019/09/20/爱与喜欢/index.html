爱与喜欢

发表于 2019-09-20 | 分类于 笔记 |
| 字数统计: 1.8k

“爱”与“喜欢”不一样，曾经的我没什么感觉，现在倒有一些心得。

实际上这篇文章的主题在九月份便已确认下，如今早已对当时的灵感有一些消化，时隔一个多月到了如今不妨就拿来记录现在的感受吧。

这段时间通过游戏相识了一个在南国的异性游戏好友。刚开始愿本着打算一起玩耍获取快乐就够了，没想到却越发不可收拾了…如今在我们双方都表达了对对方有好感之后，我居然越发疯狂想来一场虚无的网恋…这是愚蠢的，网恋我在清醒以来从来没想过要有这一天的发生，我一直在与自己作斗争，一边告诫自己不要越陷越深，一边克制自己对这个喜欢着有好感的南国异性的所作所为。网恋于我而言，是不现实，也是不真实的，我无法接受这样形同虚设的恋爱方式，而且对于我来说是一种巨大的煎熬。

实际上无关网恋，关于这个南国异性我都未曾了解的太多，却又急切的希望与对方建立一些联系，甚至于想要占取她心目中的一个位置，不过还好当初刚认识的时候，双方都比较理智，所以才没有造成这样轻浮的喜剧。仔细回想当初，不过是借着网络牵针引线，我这个人或许面对着一个在实现生活中毫无联系但存有好感的陌生人更容易放得开？从小到大，我有过两次这样的经历，基本上因为现实的原因，所以在网络上相互交流的时间较为多，在没有面对面的情况下，我居然能把控到自在，还能把控得游刃有余的…一直到如今我想借着这段经历认真反思，才清晰的认识到这个问题的本质所在，或许都是因为跟付出的感情相挂钩，所以总是先将问题牵引到感情上，而不是一些较为细节的基本问题上。如今回想前两段经历，再与如今这段经历相对比，不往有很多相似的地方：在我确定我们双方都存有对对方有好感的基础下，且因为不是现实中的面对面，借着网线我可以更为轻松的放开自我去跟对方交流与讨论，而且不带着一丝虚伪。我亦承认，我这样做的最终目的不过是为了获得对方更多的认可与对我的欢喜，正所谓不断获取希望才能获取不断的行动的动力。

而现在的我难道不就是为了那一个在对方心目中的一个位置吗，急切的想要获取到一个名份，而且这个名份确实可有可无，如果双方的好感不达到一定程度的话。为何那么急切？急切的想要这个名份？估计是跟我本人性格有很大的关系，我急于求成也不是一两天的事了…这点我一直较为苦恼，这确实是我最为致命的点，居然知道对方对我存有好感，我想这点目前就能足够了，有时不断的想要获取对方的东西越多，反而越发变得不自在了。我如今才明白，在面对男女感情之事时，有时急切的想要获取对方更多时，反而容易变得适得其反，首先最为明显的就是自己变得不像自己了，失去了自在，也就开始乱了方寸了…有时特别特别的想对方的某一处时，脑子里就会不断回望过去跟这一处相关的历史经历，然后不断的将这段时间的历史放大进行自我的催化，想念就会越发变得不可收拾。这时就不是简单的存有好感了…确确实实喜欢上了对方了。

关于爱，体现在更多的细节之上。更为无微不至的体贴，对方的事、对方的生活都如同自己的一般，不断的去跟着挖掘细节，然后才是不断的作猜测、作判断 ，时刻想着关心着对方的目前的状态。爱这字的含义是千言万语都表现不出来的，只能作体会留在心里细细品味。对我来说，爱就是将对方“变成”了自己，所以这样就能顺理成章的作关心，作体会了。爱，是与很多情愫相挂钩的，如同千丝万缕的藕断丝连，只能作体会。

回到主题上，我该如何处理与那位我存有很多很多好感的南国异性作更多了解呢…这其实对于我是个世纪难题，因为面对的人分千千万万种。喜欢与坚持喜欢其实亦然简单，不过是需要对方的认可与同样与我相互欢喜的感觉。曾经我最为擅长的换位思考在现在却又显得不堪一击..因为我没有更为恰当的参照物了…这很是较为头疼的问题，这样绝大部分只能作自我猜测了。猜测这活其实也是一件苦差事，稍微不合意，各种迷惑也随之而来了，然后感情开始动摇，内部斗争随之也在不远处了…哈哈，这其实在我看来是一件可笑的事，没想到堂堂我刘某人也有这狼狈的一天哈，哈哈哈…

面对好感，面对喜欢，面对爱意。我想还是得把控得了自我才行，得时刻保持理智才行，才不能糊里糊涂被某一虚化的美好瞬间卷进去越陷越深，无须存有过分美好的幻想，这些恐怕会起到推波助澜的反作用，这些都是对于将来有别离的一天是催化剂，无法自拔其实也是自找的，不过基于对方的感情是催化剂罢了，哈哈。

就这么办吧，把握好现在最为重要，未来的规划最好停留在一个又一个小目标上，这样才显得真实与豁达，过多过分的幻想其实也是庸人自扰…

细细反观，以上所作的不就是曾经我不为所动的恋爱观的其中一部分吗…怎么如今动摇了？我想是多种意想不到的因素和急切的心理在作祟吧，急切的背后又是什么？是想占为己有的野心，哈哈哈哈..
