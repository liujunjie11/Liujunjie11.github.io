网络心动时分经历记录

发表于 2019-12-13 | 分类于 笔记 |
| 字数统计: 1.4k

十月份到如今的十二月份，不知不觉是有南国异性陪伴的快两个月的时间了。

时间过得真快，你我从认识到相识已经快到两个月。现在到了真的要结束的时候了。一款游戏将你我的距离从南国拉到北国，沿着网线好似手拉手，显得格外亲近，我们无话不谈，我亦感到前所未有的轻松、自在与坦然，多久了？我多久没有这种感觉了？我怕是习惯一个人生活，一个人自娱太久了，或许是孤寂的太久了，想找个人找个地方说说话派遣一下情绪与孤独罢了？

还曾记得刚开始认识你的时候，我问你能陪我多久，你说很久很久，只要我不解除我们的关系，你就能一直陪着我。什么时候我们之间的距离从刚开始认识便火速拉近，因为我的猜疑和没有安全感，你我之间的关系距离才总算是能稍微停一停了的迹象。后来我不断猜疑猜疑，本来总是想快刀斩乱麻，长痛不如短痛的时候，我发现了你的单纯、天真以及善良，还有在某个夜晚发觉到你的真心？或许是吧，又或许是我自以为是了。我开始转变了我的态度，愿意慢慢去尝试了解你了，慢慢的居然对你有了好感，猜疑一天天在减退，对你的信任使得我的安全感慢慢的越来越多。总该归于生活，也总该归于平淡，你我之间的距离慢慢的开始拉远了，无话不谈到寥寥数语都有发生过。你跟我很像，习惯孤寂，但是可能也需要陪伴。你需要的陪伴用你的话说：就是一直可以陪着你的人。但我，自从有了对你的好感，却一发不可收拾，开始泛滥，开始躁动，开始影响我的生活…我发现我沦陷了，但每一次沦陷都在夜晚自救，在第二天早晨疲惫不堪的再去开始为生活工作，我发现我开始喜欢上你了。喜欢你，隔着网线，我的理智不允许我这般泛滥，但我面对你，即使是木板的屏幕，听着你的声音，看着你的文字，我不断告诫自己：坚持坚持。你的每一句对我有好感，我看到了希望，并且默念着未来如何如何，幻想着会怎么样怎么样，这些东西又这样的不知陪伴我多少个夜晚，我的理智同时又不断折磨我多少个夜晚，我太缺乏安全感了…我开始想了解你更多的关于你如何看待我的想法，我的方式带着无情，也带着担忧，我把你搞得人心惶惶，但不知你是否真的伤过心？同时我也是坐立不安，深怕错过你，不然我又会后悔，后悔像当初一样，面对喜欢的女孩却没有抓紧她们的手，直到永远失去，在回首过往中，我学会了珍惜。我这么做，反反复复这么做，每一次我明显感觉到了我们之间的变化，但只有这样，我才能真正从你口中得到我想要的答案，但事到如今，我们之间却再也回不去了。失望的叠加，直到绝望，我早已忘记了我还是会对你猜疑的这回事了，我还没有完全信任你呢，也还没有完全了解你呢，我们就要散了…你跟我又不一样，我如何才能找到继续下去的希望，我只是缺少一个答案，但我从你口中得不到。你说你的想法是你自己的，你没必要说出来，你说你一直认为自己会一个人一辈子，你还说我想怎么样就怎么样。后来你一句一句对我有好感在我快要退却的时候，一句你其实也很难过的时候，我仿佛又看到了希望，我以最快的速度整理好心情，调整好状态，只为我们能开心的过得像以前一样，我也不想让你感觉压抑，我想让你尽快了解真实的我，也能开开心心的过好每一天。你来找我，我可以放下所有的事情和工作，因为我喜欢你还没有变。面对我的质问，你开始会挽留我，直到你也绝情的不留一点余地。我的自尊，我的原则，在你面前全部崩塌了，我心高气傲的内心也因你而慢慢沦陷，我不是变得卑微了，我只是慢慢喜欢上了你。可能是你被我折腾的累了，而我也真的累了…

不想了，再想我就是狗！！！！！！

现在她的几句话我就顿时茅塞顿开了，我还想个屁！！！之前理智下来的规则原则好像就是个摆设…根本没个屁用，结果还是回到了之前的那个样子…想个屁！！再想我就是狗东西！！

亦不再强求感情这种事了，这种事我也只好去随缘罢，无法言喻。自行珍惜罢。
